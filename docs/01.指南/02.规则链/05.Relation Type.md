Relation Type是：节点与节点连接的关系类型，决定数据在规则链的路由路径选择。不同组件提供不同的连接关系，一般filter类组件提供的关系有：True/False/Failure；action类组件提供关系有：Success/Failure。

节点与节点连接的关系类型在规则链`connection.type`字段配置，示例：

``` json

  "connections": [
      {
        "fromId": "s1",
        "toId": "s2",
        "type": "True"
      }
    ]
    
```
